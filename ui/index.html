The Tracking Gap

California's Grant Information Act (AB-132, Gov Code §8333) requires the state to track post-award disbursement. We pulled both available fiscal years from data.ca.gov's CKAN API. The field is empty for every single record.

The Lead Case

Beyond the systemic gap, our DGS purchase order scan surfaced a pattern that matches the textbook signature of the 2019 Caltrans bid-rigging prosecution. We are not making accusations. We are pointing at public records.

PATTERN MATCH — WARRANTS INVESTIGATION $599,300

The $49,950 Pattern

12 purchase orders at exactly $49,950 each — $50 below the competitive-bidding threshold

100% single-buyer concentration (Cal Fire issued every one)
4 contracts issued within a single 7-day window
All 12 amounts identical to the dollar, not approximate
Possible charitable explanation: emergency procurement during active wildfire response

California's State Contracting Manual prohibits "splitting purchases" to evade competitive bidding. Public Contract Code §§ 10301-10340 require formal bidding above $50,000. The 2019 prosecution of three Caltrans procurement officials (USA v. Yong, Miller, Opp — 49-78 month sentences, $3M restitution) was built on this exact factual pattern. Whether this case is benign emergency procurement or something else is a question for the State Auditor — not a question we can answer from public records. We surface the pattern; oversight bodies determine intent.

The Pipeline

A multi-agent forensic audit pipeline. ETL → analysis → external verification → synthesis, with budget controls, caching, and a hard kill switch. Built with Bright Data, Gemini, and ProPublica.

1

Multi-Source ETL

26,907 grant records (CA Grants Portal), 50,000 purchase orders (DGS), 500 nonprofit 990s (ProPublica). All normalized, deduplicated, persisted to SQLite.

2

Heuristic Flagging

Per-source anomaly detection: just-under-threshold PO clusters, repeating exact amounts, buyer-vendor concentration, nonprofit overhead ratios, exec-comp YoY spikes.

3

Bright Data Verification

Web Unlocker bypasses anti-bot on bizfileonline.sos.ca.gov, CCLD, ProPublica, SAM.gov. SERP API runs 5-variant queries across all flagged entities. Hard budget cap.

4

LLM Synthesis

Gemini 2.5 Pro distinguishes real anomalies from data quality issues and DBA-trap false positives. Cleared cases get CLEARED with reasoning; survivors get WARRANTS INVESTIGATION.

5

Validation Pass

EIN-match sanity checks drop false positives (e.g., MADD flagged when we matched a $5K-revenue local chapter, not the national org). 102 of 104 nonprofit flags survived; 36 reached HIGH priority.

6

Transparent Reporting

Auto-generated markdown reports per round. Every finding cites public records. Dead-end log documents what was cleared, so future investigators don't repeat our work.

What We Explicitly Do Not Claim

  • ✕   We do not claim fraud has been proven

  • ✕   We do not use "fraud" as a conclusion — only as a pattern label
  • ✓   We surface patterns present in public records
  • ✓   We frame every finding as "anomaly warranting investigation"
  • ✓   We publish the methodology so anyone can verify or extend it

The path to actual recovery is the California False Claims Act qui tam process (file under seal before public disclosure). Our role is to surface candidates, not to make legal determinations.
